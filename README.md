# GAUDY – Creative Agency

GAUDY is a modern creative agency website built with **React JS** and designed to showcase professional branding, design, and web development services. It features clean UI/UX, responsive layout, and optimized SEO metadata.

- 🚀 Fully Responsive Design (Mobile & Desktop)
- 🎨 Clean UI/UX layout
- 🌐 SEO Meta Tags
- 💼 Portfolio Section
